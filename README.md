# neoputt
a web-based multiplayer minigolf simulator.

   <img src="./doc/gameplay.png" alt="gameplay screenshot">

   <img src="./doc/lobby.png" alt="online lobby screenshot">


## installation

    $ npm install neoputt

    $ cd node_modules/neoputt

    $ npm run build

    $ npm start
